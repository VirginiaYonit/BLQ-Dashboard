# 25 Years at BLQ — Dash Data App

This interactive dashboard tracks 25 years of air traffic at Bologna Airport, including:

- Passenger trends
- Cargo movements
- CO₂ emissions
- Pre-departure delays
- Load factor efficiency vs environmental impact
